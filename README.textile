h1. ParserExample

This is a project showing you how to use AQXMLParser and HTTPMessage together with AQGzip* from aqtoolkit.

h1. Building

Before building this project, you will need to retrieve the aqtoolkit submodule:

```
cd ParserExample
git submodule update --init
```
